# PictoData

Website link: https://pictodata.herokuapp.com/

## Getting started

First, you'll want to download the library file and place it into your repository (link can be found on the homepage of website).

After that, all you need to do is include the js file for PictoData and jQuery (jQuery is a requirement for PictoData).
This would look something like this:

```
<script type="text/javascript" src='PictoData.js'></script>
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
```
