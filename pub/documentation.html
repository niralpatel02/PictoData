Getting started

First, you'll want to download the library file and place it into your repository (link can be found on the homepage).

After that, all you need to do is include the js file for PictoData and jQuery (jQuery is a requirement for PictoData).

This would look something like this:
<script type="text/javascript" src='PictoData.js'></script>
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
You can now start creating your graphs using PictoData. Examples of this can be found under the 'Examples' tab.

Methods

Method Parameter Description
pictodata.bar(info) {
'Element': object,
'Data': [integer],
'Labels': [String],
'colours': [String],
'height': integer,
'width': integer
} 		Creates a bar graph using information provided by the 'info' parameter.
pictodata.bubble(info) {
'Element': object,
'Data': [integer],
'Labels': [String],
'colours': [String],
'height': integer,
'width': integer
} 		Creates a bubble chart using information provided by the 'info' parameter.
pictodata.word(info) {
'Element': object,
'Data': [integer],
'Labels': [String],
'colours': [String],
'height': integer,
'width': integer
} 		Creates a word chart using information provided by the 'info' parameter.
pictodata.addData(info) {
'Data': [integer], 'Labels': [String], 'colours': [String]
} 		Adds data to the current graph using information provided by the info parameter.
pictodata.removeData(info) Integer Removes data from the current graph using the index provided by the info parameter.
pictodata.createLegend(info) object Creates a legend for the current graph and places it in the element provided by the info parameter.
